# Portfólio - Fábio Mendes

O **Portfólio** é um site pessoal desenvolvido para apresentação profissional, detalhando minha trajetória, habilidades, formação acadêmica e projetos realizados. O projeto oferece suporte multilíngue (Português, Inglês e Espanhol) e alternância entre os modos claro e escuro.

## 🚀 Arquitetura e Tecnologias

A aplicação é um projeto frontend moderno focado em performance, acessibilidade e design responsivo.

### 1. Frontend (Interface do Usuário)
- **Framework:** Next.js 15 (App Router)
- **Linguagem:** TypeScript
- **Estilização:** Tailwind CSS e Radix UI (Componentes de UI)
- **Animações:** Framer Motion
- **Ícones:** Lucide React e Radix Icons
- **Hospedagem:** Vercel / GitHub Pages

### 2. Funcionalidades Principais
- 🌍 **Multilíngue:** Suporte nativo para PT, EN e ES através de dicionários locais (`.json`).
- 🌙 **Dark/Light Mode:** Alternância de tema fluida com persistência.
- 📱 **Design Responsivo:** Layout adaptativo para dispositivos móveis, tablets e desktops.
- ⚡ **Performance e SEO:** Renderização otimizada para os motores de busca e carregamento ultra-rápido.

## ⚙️ Pré-requisitos

Para rodar o projeto localmente, certifique-se de ter instalado em sua máquina:
- [Node.js](https://nodejs.org/) (Versão LTS recomendada)
- [pnpm](https://pnpm.io/) (Gerenciador de pacotes utilizado pelo projeto)

---

## 🛠️ Como Executar o Projeto

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/fab1omendes/fab1omendes.github.io.git
   cd fab1omendes.github.io
   ```

2. **Instale as dependências:**
   ```bash
   pnpm install
   ```

3. **Suba o servidor de desenvolvimento:**
   ```bash
   pnpm run dev
   ```

4. **Acesse a aplicação:**
   Abra [http://localhost:3000](http://localhost:3000) no seu navegador.
